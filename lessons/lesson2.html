🎯 看涨期权（Call Option）是什么？

定义

看涨期权 = 认购期权 = Call

买方支付权利金，获得在未来以约定价格买入标的资产的权利。

  • 你认为：标的价格会涨
  • 你买入：看涨期权（Call）
  • 如果涨了：行权获利
  • 如果跌了：放弃行权，损失权利金

💰 盈亏怎么算？

h3>看涨期权盈亏公式
盈亏 = (标的价格 - 行权价) × 合约单位 - 权利金

老张的案例计算：

  • 行权价：4.000元
  • 到期时510300价格：4.100元
  • 合约单位：10000份
  • 权利金：890元
盈亏 = (4.100 - 4.000) × 10000 - 890 = 1000 - 890 = +110元

但是！如果老张买的是 "510300购4月3900"（行权价3.900），结果会大不同...往下看。

💡 看涨期权核心要点

  1. 买入看涨期权（Long Call） = 看多 = 赌涨
  2. 最大损失 = 支付的权利金（最多亏完）
  3. 最大收益 = 理论上无上限（标的价格涨越多，赚得越多）
  4. 盈亏平衡点 = 行权价 + (权利金/合约单位)
  5. 时间价值：越接近到期，时间价值衰减越快

🎯 什么时候买Call？

  • ✅ 强烈看涨，但不想承担买入ETF的大额资金风险
  • ✅ 用小额资金博取杠杆收益
  • ✅ 对冲已有的ETF空头（融券卖出）
  • ❌ 横盘或看跌时买Call = 大概率亏光权利金

⚡ 实战技巧：选哪个行权价？

类型 实值期权 平值期权 虚值期权
定义（510300=3.900） 行权价< 3.900 行权价≈ 3.900 行权价> 3.900
权利金 贵（时间+内在价值） 中等 便宜（只有时间价值）
杠杆 低 中 高（但容易归零）
适合场景 稳健看涨 平衡选择 激进看涨/博暴涨
