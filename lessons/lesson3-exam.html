风险与盈亏：损益特征与计算

预计学习时间：25分钟 | 覆盖考试25%分值 | 含计算题

🎯 本课考试重点（必背！）

  1. 买方：损失有限（权利金），收益无限（认购）或有限（认沽）
  2. 卖方：收益有限（权利金），风险无限（认购）或较大（认沽）
  3. 内在价值 = Max(0, 标的价格 - 行权价) for 认购
  4. 时间价值 = 权利金 - 内在价值
  5. 盈亏平衡点 = 行权价 ± 权利金（每股）

📚 考点一：买方损益特征（必考！）

认购期权（看涨）买方

  • 最大损失：支付的权利金
  • 最大收益：理论上无限（标的价格可无限上涨）
  • 盈亏平衡点：行权价 + 每股权利金

认沽期权（看跌）买方

  • 最大损失：支付的权利金
  • 最大收益：有限（行权价×合约单位 - 权利金，标的价格最低为0）
  • 盈亏平衡点：行权价 - 每股权利金

💡 记忆口诀：买方风险有限

"最坏情况亏光权利金"

  • 买认购：赌大涨，涨越多赚越多，理论上无限
  • 买认沽：赌大跌，最多跌到0，收益有限

📚 考点二：卖方损益特征（必考！）

认购期权（看涨）卖方

  • 最大收益：收取的权利金
  • 最大风险：理论上无限（标的价格可无限上涨）

认沽期权（看跌）卖方

  • 最大收益：收取的权利金
  • 最大风险：较大（行权价×合约单位 - 权利金，标的价格最低为0）

💡 记忆口诀：卖方收益有限、风险巨大

"最好情况赚光权利金，最坏情况血本无归"

  • 卖认购：赌不涨/小涨，但万一暴涨，损失无限
  • 卖认沽：赌不跌/小跌，但万一暴跌，损失很大

📚 考点三：内在价值与时间价值（必考！）

内在价值（Intrinsic Value）

期权立即行权可以获得的收益。

认购期权内在价值 = Max(0, 标的价格 - 行权价)

认沽期权内在价值 = Max(0, 行权价 - 标的价格)

  • 实值期权：内在价值 > 0（标的价格 > 行权价 for 认购）
  • 平值期权：内在价值 = 0（标的价格 ≈ 行权价）
  • 虚值期权：内在价值 = 0（标的价格 < 行权价 for 认购）

时间价值（Time Value）

权利金中超出内在价值的部分，反映期权到期前标的价格波动带来的潜在收益。

时间价值 = 权利金 - 内在价值

  • 到期日临近：时间价值加速衰减（Theta风险）
  • 到期日当天：时间价值 = 0

计算示例

510300购4月4000，权利金680元，510300现价4.100元：

  • 内在价值 = (4.100 - 4.000) × 10000 = 1000元
  • 时间价值 = 680 - 1000 = -320元？不对！

等等，这个例子有问题！如果内在价值（1000元）已经大于权利金（680元），这个期权被低估了。

正确示例：权利金1500元：

  • 内在价值 = 1000元
  • 时间价值 = 1500 - 1000 = 500元

📚 考点四：盈亏计算（计算题必考！）

认购期权买方盈亏

盈亏 = (标的价格 - 行权价) × 合约单位 - 权利金

（当标的价格 > 行权价时）

盈亏平衡点 = 行权价 + (权利金/合约单位)

认沽期权买方盈亏

盈亏 = (行权价 - 标的价格) × 合约单位 - 权利金

（当标的价格 < 行权价时）

盈亏平衡点 = 行权价 - (权利金/合约单位)

📝 计算题示例

题目：投资者买入1手510300购4月4000，支付权利金1200元。到期日510300价格为4.150元，求盈亏。

解答：

  1. 行权收益 = (4.150 - 4.000) × 10000 = 1500元
  2. 净盈亏 = 1500 - 1200 = +300元
  3. 盈亏平衡点 = 4.000 + (1200/10000) = 4.120元

📝 开户考试真题练习

1. 【单选】认购期权买方最大收益为____，最大损失为____。

A. 无限；权利金
B. 权利金；无限
C. 无限；无限
D. 权利金；权利金

2. 【单选】认沽期权卖方最大风险为____。

A. 无限
B. 较大（行权价×合约单位 - 权利金）
C. 权利金
D. 0

3. 【计算】投资者买入1手510300购4月4000，权利金800元。到期日510300价格为4.120元，盈亏为____元。

A. +800
B. +400
C. -400
D. -800

4. 【单选】期权的时间价值是指____。

A. 权利金 + 内在价值
B. 权利金 - 内在价值
C. 内在价值 - 权利金
D. 标的价格 - 行权价

5. 【判断】认购期权买方盈亏平衡点 = 行权价 - 权利金。（ ）

A. 正确
B. 错误（应为行权价 + 权利金）

6. 【多选】关于期权卖方，以下说法正确的是（ ）。

A. 最大收益为权利金
B. 认购期权卖方风险理论上无限
C. 最大损失为权利金
D. 认沽期权卖方风险较大（非无限）
