## 技术栈升级

### 当前
- HTML + CSS + JS（静态）
- 无框架
- 无后端

### 目标
- 前端：Vue 3 + TypeScript + Vite
- UI库：Element Plus / Ant Design Vue
- 状态管理：Pinia
- 后端：Node.js + Express（后续）
- 数据库：MongoDB（后续）
- 部署：Vercel / 腾讯云
