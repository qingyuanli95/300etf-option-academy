# 300ETF期权学院

🎯 **零基础学懂510300沪深300ETF期权**

> 一个专为期权新手设计的互动学习网站，以510300沪深300ETF期权为实战标的。

## 📚 课程内容

### 学习路径

| 关卡 | 内容 | 特色 |
|------|------|------|
| 第零关 | 认识510300 | 了解标的、合约规则 |
| **第一课** | 权利的秘密 | 用买房故事理解期权本质，510300盈亏计算器 |
| **第二课** | 看涨期权实战 | 买CALL赌上涨，完整盈亏图表 |
| **第三课** | 看跌期权实战 | 买PUT赌下跌/对冲，对比演示 |
| **第四关** | 实战模拟器 | 10天模拟交易，用虚拟资金实战 |
| **第五课** | 策略实验室 | 保护性看跌、备兑开仓、跨式、价差策略 |
| **第六课** | 隐含波动率 | IV定价原理、IV Crush、高/低IV策略 |

### 核心功能

- ✅ **互动演示**：拖动滑块实时计算盈亏
- ✅ **图表可视化**：直观展示盈亏曲线
- ✅ **知识测验**：每课结束有测验检验学习效果
- ✅ **模拟交易**：零风险体验真实期权交易

## 🚀 快速开始

1. 在浏览器中打开 `index.html`
2. 点击"开始实战之旅"
3. 按顺序完成课程学习
