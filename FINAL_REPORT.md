## 📱 技术栈

### 当前技术
- **前端**: HTML5 + CSS3 + Vanilla JS
- **样式**: 自定义CSS框架 + 响应式设计
- **存储**: LocalStorage (离线可用)
- **动画**: CSS3 + Canvas
- **PWA**: Web App Manifest

### 下一阶段技术升级
- **框架**: Vue 3 + TypeScript
- **构建**: Vite
- **UI库**: Element Plus
- **状态**: Pinia
- **后端**: Node.js + Express (可选)
